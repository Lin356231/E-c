## Overview

This project is a complete e-commerce solution designed for small to medium businesses. It includes a front-end for customers, a secure shopping cart and payment system, and an admin panel to manage products, orders, and users.

Ideal for learning full-stack development or launching an online store.

## Features

- User authentication and profile management
- Browse, search, and filter products
- Shopping cart with quantity controls
- Secure checkout with payment gateway integration
- Admin panel to manage products, orders, and users
- Responsive UI for mobile and desktop

## Installation

Clone the repository and set up the environment:

```bash
git clone https://github.com/Lin356231/E-c.git
cd e-c
````

**Backend setup:**

```bash
cd backend
npm install
# Set environment variables in .env
npm run dev
```

**Frontend setup:**

```bash
cd frontend
npm install
npm start
```

## Usage

1. Register as a new user or login with existing credentials.
2. Browse products and add items to your cart.
3. Proceed to checkout and place your order.
4. If you’re an admin, login to the admin panel to manage inventory and view orders.

## Technologies Used

* **Frontend:** React, Redux, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MongoDB with Mongoose
* **Authentication:** JWT, bcrypt
* **Payment:** Stripe API
* **Deployment:** Vercel (frontend), Render/Heroku (backend)
